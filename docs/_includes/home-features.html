Build Interfaces

Creating UIs and GUIs for your Sketch Plugin can be a pain, even more when you try to do it with plain JS / Cocoascript. This boilerplate gives you useful helpers to achieve that goal.

Async HTTP Requests

You need to do API requests from your plugin? The boilerplate comes with helpful utils and a fetch polyfill that make asynchronous network request and downloads super easy.

Get Up and Running

If you are tired of setting up projects from skretch you came to the right place. This boilerplate comes with a comprehensive setup, so you can start doing the fun stuff straight away.
